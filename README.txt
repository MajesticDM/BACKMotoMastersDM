---------------------------Moto Masters MD----------------------------------------------------------

Proyecto creado para uso comercial o educativo por Daniel Moreno. La principal idea de este proyecto es aprender a crear una Api (REST Full) para aplicaciones, ya sean web o móbiles, ya que es fácilmente entendible y puede ser editada sin problemas ya que cuenta con un código amigable.
Cabe recalcar que está en inglés debido a que permite un mejor entendimiento con el ambiente de programación, y así no tener que estar escribiendo código en español y las funciones en inglés.

Siéntete libre de usar este código a gusto, ya que ese es su objetivo.
